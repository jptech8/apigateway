## Installation

```bash

#Windows
set NODE_ENV = dev [eg. dev , prod,preprod,qa] 

#Linux
export NODE_ENV =dev [eg. dev , prod,preprod,qa] 
```
Use the package manager NPM to install dependencies in project folder.
```bash
npm install
```

```bash
npm start
```
Only for local development 
```bash
npm run local
```
